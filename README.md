# HelpDesk — Система регистрации заявок в службе технической поддержки

## Стек технологий
- Python 3.8+ / Flask 3.0
- SQLite (через Flask-SQLAlchemy)
- Bootstrap 5 + Jinja2
- PWA (работает на Android и iOS)

---

## Быстрый старт

### 1. Установить зависимости
```
pip install -r requirements.txt
```

### 2. Запустить приложение
```
python app.py
```

### 3. Открыть в браузере
```
http://localhost:5000
```

При первом открытии сайта откроется страница первоначальной настройки —
создайте учётную запись администратора (имя, email, пароль).

---

## Структура проекта

```
helpdesk/
├── app.py                    # Маршруты и логика обработки запросов
├── models.py                 # Модели базы данных (User, Ticket)
├── config.py                 # Настройки приложения
├── requirements.txt          # Зависимости Python
├── helpdesk.db               # База данных SQLite (создаётся автоматически)
│
├── templates/
│   ├── base.html             # Базовый шаблон (навигация, PWA)
│   ├── login.html            # Страница авторизации
│   ├── setup.html            # Первоначальная настройка (создание admin)
│   ├── index.html            # Список заявок
│   ├── new_ticket.html       # Форма создания заявки
│   ├── ticket.html           # Просмотр заявки
│   ├── search.html           # Поиск по исполнителю и дате
│   ├── users.html            # Список пользователей (admin)
│   └── new_user.html         # Создание пользователя (admin)
│
└── static/
    ├── manifest.json         # PWA манифест
    ├── service-worker.js     # PWA Service Worker (кеширование)
    ├── icon-192.png          # Иконка приложения 192×192
    └── icon-512.png          # Иконка приложения 512×512
```

## Роли пользователей

| Роль    | Права                                                    |
|---------|----------------------------------------------------------|
| user    | Создавать заявки, просматривать свои заявки              |
| support | Менять статус заявок, поиск, просматривать свои заявки   |
| admin   | Всё + назначение исполнителей, управление пользователями |

## Установка PWA на смартфон

### Android (Chrome)
1. Открыть `http://[IP-сервера]:5000` в Chrome
2. Нажать меню (три точки) → «Установить приложение»
3. Иконка HelpDesk появится на главном экране

### iOS (Safari)
1. Открыть `http://[IP-сервера]:5000` в Safari
2. Нажать кнопку «Поделиться» ⬆
3. Выбрать «На экран "Домой"» → «Добавить»

> ⚠️ На iOS PWA устанавливается только через Safari.

## Доступ из локальной сети

Узнать IP-адрес сервера:
- Windows: `ipconfig`
- Linux/Mac: `ip addr`

Открыть с телефона: `http://[IP-адрес]:5000`

---

## Доступ через интернет (с мобильного интернета)

Быстрый способ — ngrok:
```
ngrok http 5000
```
Ngrok выдаст публичный адрес вида `https://abc123.ngrok-free.app` —
он открывается с любого устройства через мобильный интернет.

Для постоянного адреса — арендуйте VPS (~200-400 руб/мес).
